# Test Strategy

Automated tests are not yet implemented.

## Recommended Baseline
- Add PHPUnit for unit tests of helper functions.
- Add integration tests for auth and role-protected routes.
- Add smoke tests for critical pages (viewer, author dashboard, admin dashboard).

## Manual Regression Checklist
- Login/logout flow
- Author create/edit/publish flow
- Viewer post listing/detail pages
- Admin user role updates
